# Get-TerminalIconsColorTheme

## SYNOPSIS
List the available color themes.

## SYNTAX

```
Get-TerminalIconsColorTheme
```

## DESCRIPTION
List the available color themes.

## EXAMPLES

### EXAMPLE 1
```
Get-TerminalIconsColorTheme
```

Get the list of available color themes.

## PARAMETERS

## INPUTS

### None.
## OUTPUTS

### System.Collections.Hashtable
### An array of hashtables representing available color themes.
